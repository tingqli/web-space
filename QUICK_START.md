## ⚡ 核心概念 (3分钟理解)

### 问题: 如何表示汇编指令对应的源代码行号?

### 答案: DWARF Line Table

```
单个字节 = 行增量 + 地址增量
13-255  行[-5,8]  地址[0,17]

例: opcode=47 → 行+1, 地址+2
```

### Inline (另一个问题: 如何表示inline调用链?)

```
DIE Tree:
  main函数
    └─ inlined_subroutine (inline_func被inline的实例)
        ├─ DW_AT_abstract_origin → 指向inline_func原型
        ├─ DW_AT_call_line → 调用点行号
        └─ DW_AT_call_column → 调用点列号
```
