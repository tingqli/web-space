MLIR Loc Explorer

一个用于 VS Code 的双栏对照插件：基于 MLIR loc 信息进行跨文件联动定位、锁定高亮和源码跳转，帮助你快速追踪编译前后语句关系。

下载 VSIX 如何安装 VSIX

Loc 联动高亮

左右文件按 loc 自动匹配，悬停即可看到对应行，快速定位中间表示关系。

锁定与持续匹配

单击锁定后切换任一侧文件，会尽量保留未切换侧锁定并重建高亮。

Python 源码隐含 Loc

左侧选择 .py 文件时，按行号隐含映射为 loc(path:line:0)，可与 MLIR 或汇编文件联动高亮。

汇编 .s Loc 解析

支持解析 .file 与 .loc 伪指令，把汇编代码行映射回高级语言源码位置，实现与 MLIR 同级联动。

源码跳转

Ctrl + 点击 某一行可在 VS Code 中打开命中的 loc 源文件并跳转到行列位置。

本次更新能力

  • 打开目录时会扫描 00_ 开头的 mlir 文件，提取 loc 中的源码路径并优先放到左侧列表
  • 左侧默认优先选中提取出的源码文件，同时保留同目录其他 mlir 文件可选
  • .py 文件支持基础语法高亮，并参与悬浮匹配、点击锁定与跨侧联动高亮
  • .s 汇编文件支持基于 .file 和 .loc 的 loc 解析与跨侧联动高亮

快速安装

  • 下载上面的 .vsix 文件
  • 在 VS Code 中执行 Extensions: Install from VSIX...
  • 选择下载后的文件并安装
